## RULES.md

Rule 1:
Filesystem is the source of truth.

Rule 2:
Original files must never be modified.

Rule 3:
Derived data is disposable.

Rule 4:
Reindexing must always be possible.

Rule 5:
Retrieval is mandatory.

Rule 6:
The entire library must never be loaded into model context.

Rule 7:
Components must remain replaceable.

Rule 8:
The catalog must remain organized even when the filesystem is not.

Rule 9:
Librarian stores knowledge about books, not the books themselves.

Rule 10:
Use the cheapest retrieval mechanism capable of answering the question.
